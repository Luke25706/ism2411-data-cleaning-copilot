This project loads a raw sales CSV file, cleans the data, and saves a cleaned version into the data/processed folder. The script fixes column names, cleans categories and product names, handles missing or invalid values, removes duplicates, and converts the date column.

How to run:
	1.	Open a terminal in the project folder.
	2.	Run: python src/data_cleaning.py
	3.	The cleaned file will appear in data/processed/sales_data_clean.csv
